Question-1: What are the different ways to select an element in the DOM?

Answer: There are many ways but mostly used practices are: 1.By Id, 2.By Class, 3.By Tag, 4.By CSSselector,5.By Name, 6.By Attribute and etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Key Differences:
i.innerHTML: Includes HTML tags and hidden text.
ii.innerText: Excludes HTML tags and hidden text, respects CSS.
iii.textContent: Excludes HTML tags but includes hidden text.
Use innerHTML for HTML manipulation, innerText for visible text, and textContent for all text content.

Question-3: What is event delegation in the DOM?

Answer: Event Delegation is a technique in JavaScript where instead of adding event listeners to each individual element, you add a single event listener to a parent element. This parent element listens for events that bubble up from its child elements.

Question-4: What is event bubbling in the DOM?

Answer: Event Bubbling is a behavior in the DOM where an event triggered on a child element "bubbles up" through its parent elements in the DOM tree. This means the event is first handled by the child, then by its parent, and so on, all the way up to the root of the document.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer: In JavaScript, one can dynamically manipulate the DOM:
i.Create: Use document.createElement('tag') to create a new element.
ii.Add: Use parent.appendChild(newElement) to add it as the last child, or parent.insertBefore(newElement, referenceElement) to insert it before a specific child.
iii.Remove: Use parent.removeChild(child) or child.remove() to delete an element.
These methods enable dynamic DOM manipulation.
